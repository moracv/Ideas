# Ideas 💡

A PWA (Progressive Web App) for managing and analyzing your ideas with AI-powered insights.

## Features

- 📝 Create and organize your ideas
- 🤖 AI-powered analysis and viability scoring
- 📊 Visual insights and statistics
- 🔄 Sync with Google Sheets
- 🎨 Clean and intuitive interface

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding your ideas!

## Technologies

- HTML5
- JavaScript (Vanilla)
- Chart.js for visualizations
- Google Gemini AI integration
